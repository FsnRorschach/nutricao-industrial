# Nutrição Industrial

Landing page profissional para consultoria em nutrição industrial, criada para apresentar serviços, depoimentos, formulário de contato, botão de WhatsApp e modal de aviso "em desenvolvimento".

## Sobre o projeto

Este projeto é uma página estática voltada para nutricionistas e consultores de nutrição industrial. A proposta é oferecer um layout moderno e responsivo que destaca serviços como rotulagem, HACCP e reformulação de produtos, ao mesmo tempo que facilita o contato direto com clientes via WhatsApp.

## Funcionalidades

- Cabeçalho hero com imagem de fundo e chamada de ação.
- Seções de apresentação de serviços, resultados e depoimentos.
- Formulário de contato integrado com Formspree.
- Botão flutuante de WhatsApp.
- Botão flutuante para voltar ao topo da página.
- Modal de aviso "Em Desenvolvimento" acionado por JavaScript.
- Estilos com Tailwind CSS via CDN e custom CSS local.
- Estrutura simples e fácil de customizar.

## Estrutura do projeto

- `site/index.html` - Página principal do site.
- `site/assets/css/styles.css` - Estilos personalizados do site.
- `site/assets/js/main.js` - Script de carregamento do modal e funções globais.
- `site/assets/js/modal.js` - Funções de abertura e fechamento do modal.
- `site/assets/images/hero.png` - Imagem de fundo do hero.
- `site/assets/images/foto-profissional.png` - Imagem de apresentação profissional.

## Tecnologias usadas

- HTML5
- CSS com Tailwind CSS via CDN
- JavaScript ES Modules
- Font Awesome para ícones

## Como usar

1. Abra `site/index.html` em um navegador.
2. Edite os textos e contatos diretamente no HTML.
3. Atualize o número do WhatsApp nos links para o seu contato.
4. Substitua a imagem `assets/images/hero.png` e `foto-profissional.png` pelos seus recursos visuais.

## Customização recomendada

- Ajustar o título e a descrição do projeto em `site/index.html`.
- Configurar o Formspree com o seu endpoint no formulário de contato.
- Atualizar os valores dos indicadores e depoimentos.
- Personalizar cores e tipografia usando classes Tailwind.
